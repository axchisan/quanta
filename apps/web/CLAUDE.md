# @quanta/web — Next.js 15 Frontend

## Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- shadcn/ui
- TanStack Query
- Zustand

## Scripts
```bash
pnpm dev         # localhost:3000
pnpm build       # Static export
pnpm typecheck   # tsc --noEmit
pnpm lint        # ESLint
```

## Convenciones
- Components en `src/components/`
- App routes en `src/app/`
- API routes en `src/app/api/`
- Componentes shadcn en `src/components/ui/`
