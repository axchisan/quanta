# @quanta/types — Shared TypeScript Types

## Stack
- TypeScript
- Zod (validación runtime)

## Scripts
```bash
pnpm build       # tsc
pnpm typecheck   # tsc --noEmit
```

## Contenido
- Tipos de dominio (Challenge, Room, Player, etc.)
- Tipos de API (requests/responses)
- Tipos de Supabase (generados de DB)
- Schemas Zod para validación
